# HR-Analytics-Dashboard
HR Analytics Dashboard – MySQL + Power BI

This project analyzes employee attrition and workforce patterns using the IBM HR dataset. The dataset was first cleaned and processed in MySQL and then visualized using Power BI to create an interactive HR analytics dashboard.

Key Objectives

• Identify employee attrition trends
• Analyze workforce distribution across departments and roles
• Evaluate salary patterns and demographic factors affecting attrition

Tools & Technologies

• MySQL – Data cleaning and preprocessing
• Power BI – Data modeling and visualization
• DAX – KPI calculations and metrics

Dashboard Features

• Attrition rate analysis
• Department and job role breakdown
• Age group and gender distribution
• Salary analysis by job role
• Tenure-based attrition insights
